A beautiful, romantic apology website built with HTML, CSS, and JavaScript.

## ✨ Features

- **10 Beautiful Pages** with smooth scrolling animations
- **Loading Screen** with elegant spinner
- **Navigation Bar** to jump to any section
- **Progress Bar** showing scroll progress
- **Falling Hearts Animation** with romantic emojis
- **Photo Gallery** for shared memories
- **1000 Sincere Apologies** generated dynamically

## 🚀 How to Use

1. Open the `index.html` file in any web browser
2. Click the music button (🎵) to play background music
3. Scroll through all 10 pages
4. Customize the content in the HTML file as needed

## 🎨 Customization

### Add Your Photos
Edit the `photoUrls` array in the script section:
```javascript
const photoUrls = [
    'your-photo-url-1.jpg',
    'your-photo-url-2.jpg',
    'your-photo-url-3.jpg',
    'your-photo-url-4.jpg'
];
```

### Change Background Music
Replace the audio source URL:
```html
<source src="your-music-url.mp3" type="audio/mpeg">
```

### Edit Text Content
Simply modify the text in each `<section>` to personalize your message.
